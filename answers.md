# 3.6 transformer LM resource accounting 
>(a)以GPT-2 XL为例，其配置参数如下：
词汇表大小：50257; 上下文长度：1024; 层数：48; 模型维度：1600; 
注意力头数：25; 前馈网络维度：6400; 假设我们采用该配置搭建模型，那么模型会拥有多少可训练参数？若每个参数均采用单精度浮点数存储，仅加载该模型需要占用多少内存？

参数量：
1. 50257 * 1600(词嵌入层) + 1024 * 1600(位置嵌入层，注意GPT2用的是可训练的绝对位置嵌入层而不是RoPE)
2. 单个transformer block: 1600 * 1600 * 4(QKVO投影) + 1600 * 6400 * 2(FFN) + 1600 *2 * 2(2 个LN，每个含 scale + bias)
3. 最后输出层：一个LN+一个投影（复用词嵌入层）: 1600 * 2

计算结果：50257 * 1600 + 1024 * 1600 + (1600 * 1600 * 4 + 1600 * 6400 * 2 + 1600 * 2 * 2)*48 + 1600 * 2 = 15.57e08

模型参数大小：15.57e08 * 4 / 1024 / 1024 / 1024 = 5.8GB
